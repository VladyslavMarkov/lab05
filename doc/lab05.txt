Завдання №13: Знайти максимальне просте число, що належить діапазонуь [a,b] 

Основна частина:
- опис роботи: Просте число - це число, яке має тільки два дільники це (1 і саме число). Потрібно вивести найбільше просте число з діапазону 1-20.

----------1(while)---------------------------------------------------------

#include <stdio.h>

int main()
{
   
   int a, b, i, max;//Задаємо діапозон та макісмальне число
   
   a = 20;
   b = 1;
   max = 2;
   
   while(a > b)//Запуск циклу для перевірки чисел у встановленому діапазоні
   {
       b++;
       i = 2;
       while(b%i != 0) //Запуск циклу для того, щоб дізнатися число просте чи ні
       {
           i++;
       }
       if(b == i && b > max)
       {
          max = b;      
       }      
   }
      return 0;
}
